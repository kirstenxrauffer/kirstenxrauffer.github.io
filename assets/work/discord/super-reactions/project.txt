Nitro feature that fires an animated burst effect when reacting to a message, visible to everyone in the chat. Transforms a standard reaction into a full-screen expressive moment, across both desktop and mobile.

Animations were built with Lottie. To make every super reaction feel personal to the emoji that triggered it, we extracted the top colors from each emoji and piped them directly into the animation, so every reaction feels tailor-made rather than templated.

skills: React · TypeScript · Animation · Emoji · Lottie · DataDog · Figma · Accessibility
