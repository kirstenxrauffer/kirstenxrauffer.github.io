Desktop-side implementation of Discord's custom app icon Nitro perk, accessible via User Settings → Appearance → App Icon. Changes the icon displayed within the app on Windows and macOS; expands with seasonal drops like Halloween.

origin: Pitched directly to head of product while the team was down a PM and designer. Owned the full feature end-to-end: designed the settings UI, user experience, and marketing; adapted the mobile icon assets for desktop without support from the core art team; developed it in a single sprint; and launched it.

skills: React · TypeScript · Desktop · Product Design · Asset Production · Marketing · Leadership · DataDog · Figma · Accessibility

proud of:
- Took the feature from pitch to shipped with no PM or designer. Covered product, design, and engineering myself.
- Adapted all mobile icon assets for desktop independently, without pulling in the core art team
- Fought internally for users to keep their chosen icon even after Nitro expires. Confirmed working in the wild: reddit.com/r/discordapp/comments/1kmcpdk/
- Seasonal icon drops tied to Discord moments (Halloween, etc.) turned the feature into a recurring content touchpoint: instagram.com/p/Cx8emyEKuXD/

settings path (desktop): User Settings → Appearance → App Icon
